# Documentacao

Este diretorio concentra documentos auxiliares do projeto.

## Swagger/OpenAPI

A documentacao interativa da API esta disponivel quando a aplicacao estiver em execucao:

```text
http://localhost:3000/api-docs
```

Se a variavel `PORT` estiver configurada no `.env`, use a porta definida nela.

## Arquivos relacionados

- `src/docs/swagger.js`: especificacao OpenAPI e configuracao do Swagger UI.
- `README.md`: guia principal do projeto.
